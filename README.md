# AI UX Research Report Generator

A modern enterprise dashboard for generating UX research reports using AI.

## Workflow
1. **Research Input**: Upload or paste research data.
2. **AI Analysis**: Process data and extract key findings.
3. **Insight Map**: Visualize clusters and ranking.
4. **Personas**: Generate user archetypes.
5. **Opportunity Map**: Prioritize features (Impact vs Effort).
6. **Report**: Final preview and export.

## Tech Stack
- React
- Recharts
- Framer Motion
- Lucide Icons
- Vanilla CSS
